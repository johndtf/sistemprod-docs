# Arquitectura del sistema

## Frontend

Tecnologías:

- HTML
- CSS
- JavaScript

Estructura principal:

frontend/
├── html/

## Backend

Tecnologías:

- Node.js
- Express

Estructura principal:

backend/
├── controllers/
├── routes/
├── middleware/
├── db.js
├── app.js

Actualmente las consultas SQL se encuentran dentro de los controladores.
