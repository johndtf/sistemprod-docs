# SISTEMPROD

Sistema de gestión para planta de reencauche de llantas.

## Objetivo

Controlar el proceso completo de reencauche desde la recepción de la llanta hasta la entrega al cliente.

## Tecnologías

### Frontend

- HTML
- CSS
- JavaScript Vanilla

### Backend

- Node.js
- Express

### Base de datos

- MySQL

## Módulos principales

- Clientes
- Órdenes
- Producción
- Ventas
- Consultas
- Administración

## Estado actual

Proyecto en desarrollo.

Actualmente se encuentran implementados:

- Gestión de clientes
- Gestión de órdenes
- Gestión de llantas
- Sistema de autenticación
- Navegación por pestañas
- Alertas personalizadas (Toast)
- Estructura inicial de Producción
